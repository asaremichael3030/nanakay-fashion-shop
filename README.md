# nanakay-fashion-shop
A modern, elegant fashion e-commerce website built with HTML, CSS, and JavaScript.
Designed for luxury shopping experiences with a responsive layout, product filtering, and smooth UI interactions.

✨ Live Preview
https://nanakfashionboutique.netlify.app/

 Features
🛒 Fully responsive fashion store UI
🎨 Elegant luxury design with dark/light mode support
🧾 Product grid with hover effects and badges
🔍 Search and filter functionality
📱 Mobile-friendly navigation (hamburger menu)
🧑‍💻 Product detail layout with image gallery
💬 Newsletter subscription section
📦 Easy-to-edit product system (no backend required)
⚡ Fast, lightweight, and beginner-friendly

Technologies Used
HTML5
CSS3 (Custom variables + Flexbox + Grid)
JavaScript
Font Awesome Icons
Google Fonts

📂 Project Structure
/project-folder
│── index.html
│── styles.css (inside <style> tag in your file)
│── script.js (inside <script> section)
│── assets/
├── images/

⚙️ How to Run Locally
Open index.html in your browser:
Double click it
OR
Use Live Server (VS Code extension)

🛠️ Customization Guide
This project is designed to be beginner-friendly and easy to edit:

🎨 Change Colors
Edit CSS variables:
--gold: #C9A84C;
--bg-page: #0F0F0F;
🛍️ Add / Edit Products

Find the PRODUCT DATA section in JavaScript and update:
{
  id: 1,
  name: "Product Name",
  price: 120,
  image: "image.jpg"
}
